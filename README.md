# Number Guessing Game

A simple Python-based Number Guessing Game where the computer generates a random number and the player tries to guess it.

## Features

- Generates a random number between 1 and 100
- Allows unlimited attempts
- Provides hints:
  - Too High
  - Too Low
- Counts the number of attempts
- Ends when the correct number is guessed

## Requirements

- Python 3.x

## How to Run

1. Clone the repository:

   git clone https://github.com/ItsAdityaC/number-guessing-game.git

2. Navigate to the project folder:

   cd number-guessing-game

3. Run the program:

   python guessing_game.py
